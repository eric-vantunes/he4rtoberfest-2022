**Como rodar o desafio**: 

Dentro do diretório do projeto, utilize o comando abaixo: 
```bash
ocaml main.ml SEGUNDOS
```

**Compilando a solução**

Você pode compilar a solução com:
```bash
ocamlopt -o conversor main.ml
```

E executar com:
```bash
./conversor SEGUNDOS
```
